Usage


                
Now, from your template, you can access the cookie() method to set a cookie value and the cookies object to retrieve a cookie value.


                Assuming you left the defaults in place in the above code, from an EJS template you could do the following:
                
<% cookie('myCookie3', 'My Cookie Value') %>
<!DOCTYPE html>
<html>
    <body>
        MyCookie1 = <%= cookies.myCookie1 %>
        <br />
        myCookie2 = <%= cookies.myCookie2 %>
        <br />
        myCookie3 = <%= cookies.myCookie3 %>
    </body>
</html>
                


                
Generally speaking, you'd never want to set any values from within your templates. Templates should really just be reading values. That being said, using the cookie() method from within a template would be a no-no.
